# 升级日志

## 注

新版本升级了底层 nwjs 平台，内置数据库不与老版本兼容，建议在升级前进行数据导出备份，或者数据库文件备份，防止升级失败。

数据库文件路径

```
Windows: %LOCALAPPDATA%/storywriter/
Mac: ~/Library/Application Support/storywriter/
Linux: ~/.config/storywriter
```
